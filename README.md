# MC_PC25-26
Simulación y visualización de Cadenas de Markov (MC's)

El objetivo es simular y visualizar trayectorias de una MC, usando algunas de las herramientas más habituales de machine learning (Python, Numpy, matplotlib). Como resultado final, haremos simulaciones y generaremos imágenes relacionadas con los dos teoremas de convergencia que hemos visto en clase: el teorema de convergencia de MCs y el teorema ergódico.
